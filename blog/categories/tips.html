LOAD DATA into mySQL.

Post by FV, Mar 9, 2011.

In some cases, you need to load data into mysql from a command line interface. There is a simple way to do it (linux): 


mysql -e "LOAD DATA LOCAL INFILE '$PWD/myfile.csv' INTO TABLE mytable FIELDS TERMINATED BY ','" mydb -u myuser -p
